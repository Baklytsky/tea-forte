# Button

Here, we have the `<hh-button>` atomic component in it's three main themes. In the "Canvas" section you can configure the button through both controls and CSS Variables. Those settings will effect how the snapshots below render.

> Note: As you click through the 3 themes you'll notice that the default CSS variables will apply to all themes if "Canvas" is in the "CSS Custom Properties" tab.

Here's an example configuration for each of the themes:

```css
hh-button {
  --hh-color: var(--hh-color-day);
  --hh-background: var(--hh-color-night);
  --hh-font: var(--hh-body-font);
  --hh-padding: var(--hh-spacing-1) var(--hh-spacing-2);
}

hh-button[theme='secondary'] {
  --hh-background: transparent;
  --hh-color: var(--hh-color-night);
  --hh-border-color: var(--hh-color-night);
}

hh-button[theme='tertiary'] {
  --hh-color: var(--hh-color-night);
  --hh-text-transform: uppercase;
  --hh-border-bottom-offset: -3px;
  --hh-padding: 0;
}
```

# Global CSS Vars

Here, you'll find the global CSS Vars that are injected into every Storybook Canvas. There are designed to implement the base system for a website, and be consumed when setting the CSS Variabels for any particular Web Component, like so:

```css
hh-button {
  --hh-background: var(--hh-color-night);
  --hh-color: var(--hh-color-day);
  --hh-font: var(--hh-body-font);
  --hh-text-transform: uppercase;
}

hh-button[theme='primary'] {
  --hh-background: var(--hh-color-brand-primary);
  --hh-color: var(--color-night);
}

hh-button[decoration='rounded'] {
  --hh-border-radius: 3px;
}

hh-button:hover {
  --hh-background: var(--color-night);
  --hh-color: var(--hh-color-brand-primary);
}

/* If you need, you can set non-variable overrides */
hh-button[width='full'] {
  width: 100%;
}

/* breakpoints.md */
@media screen and (min-width: 768px) {
  hh-button {
    --hh-padding: 0.5em 1em;
  }
}
```

# Button Wrapper

Here, we have the `<hh-button-wrapper>` atomic component in it's three main themes. In the "Canvas" section you can configure the button through both controls and CSS Variables. Those settings will effect how the snapshots below render.

We use this button wrapper component to render a button slotted as a child, styling to keep it consistant with the global style system. This is a useful addition to the `hh-button` component for submit buttons, since submit events do not bubble from a shadow DOM up to the root tree which can cause conflicts with form submissions.

Usage example:

```html
<hh-button-wrapper theme="secondary">
  <button type="submit" value="submit">Submit</button>
</hh-button-wrapper>
```

> Note: As you click through the 3 themes you'll notice that the default CSS variables will apply to all themes if "Canvas" is in the "CSS Custom Properties" tab.

Here's an example configuration for each of the themes:

```css
hh-button-wrapper {
  --hh-color: var(--hh-color-day);
  --hh-background: var(--hh-color-night);
  --hh-font: var(--hh-body-font);
  --hh-padding: var(--hh-spacing-1) var(--hh-spacing-2);
}

hh-button-wrapper[theme='secondary'] {
  --hh-background: transparent;
  --hh-color: var(--hh-color-night);
  --hh-border-color: var(--hh-color-night);
}

hh-button-wrapper[theme='tertiary'] {
  --hh-color: var(--hh-color-night);
  --hh-text-transform: uppercase;
  --hh-border-bottom-offset: -3px;
  --hh-padding: 0;
}
```

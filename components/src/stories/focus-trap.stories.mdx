# Focus Trap

Here we have a wrapper component that automatically traps focus when enabled.

Usage:

```html
<hh-focus-trap active="false || true">
  <div>
    <button>Button One</button>
    <button>Button Two</button>
    <button>Button Three</button>
  </div>
</hh-focus-trap>
```

### Simple Usage:

<Canvas>
  <Story name="Default">
    {() =>
      html`<button

          style="display: inline-block; margin-bottom: 10px"

          document.getElementById('trigger').addEventListener('click', (e) => {

          })
        </script>`
    }
